# 知犀思维导图使用文档
## 📖产品简介
知犀是一款全平台思维导图和思维图示软件，轻颜易用，可多人协作，理思路/记灵感/做规划/写笔记，多端云同步，节点无限制，安装完成即可在插件/手机/平板/电脑上快速制作思维导图和做大纲笔记，并支持导出PDF/高清图片/Word/Excel/Txt等格式文件。在用知犀创作思维导图笔记时，您可专注于思考和思维梳理，而不用花时间研究软件功能操作。  
知犀团队在思维导图结构上进行了部分创新，全面开放了 17 种脑图结构，向右逻辑图，思维导图、时间轴、向下分类图、桥形图、鱼骨图、组织结构图、括号图、气泡图、圆圈图、因果图等创新的 Thinking Map 图形结构，非常适用于日常的工作与学习生活，比如读书笔记、知识点梳理、产品规划、会议记录、头脑风暴、工作计划、OKR、旅游计划、考公/考研/考证类学习笔记等。

## ✏️功能简介
1. 功能多，个性化创作
知犀提供了丰富的产品功能，可以流畅插入关联线、概要、外框、图片、图标、表格等，以及自定义设置节点的形状、填充色、字体颜色、连接线结构，还可以自由调整画布的背景色、画布背景图片、思维导图风格、手绘风格，基于知犀的众多功能，人人都可以创作出漂亮又精美的思维导图。
2. 手机和电脑，设备切换畅用
借助知犀思维导图App端和网页版、电脑客户端的账号同步能力，您可以在一处制作导图，随处继续编辑和浏览。比如在电脑上编辑好的思维导图，在回家的路上还可以在手机或平板上的知犀App继续查看或者编辑，云端实时同步，保障数据不丢失。
3. 软件易上手，颜值高
知犀的小清新颜值能够让人心情愉悦，简单的流程能助您专心思考，新手操作基本不需要学习，知犀模板中心还提供了 20000+ 免费思维导图模板和知识笔记类思维导图，一般 1分钟左右即可画出简单又漂亮的思维导图。
4. 表达效果好，分享安全高效
自己的想法与他人分享时，聊天发送容易凌乱，发错了不容易撤回。若先用知犀创建思维导图来表达想法，逻辑更清晰，再使用『限时加密分享』这个功能，分享内容的隐私和时效性更有保障，在工作上分享自己的规划、总结等各类工作思维导图时，效率明显提高，且支持同步实时更新。
